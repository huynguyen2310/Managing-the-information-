# Managing-the-information-
Create username and password for admin, if you type wrong username or password then access denied.
If it is trure, you have to enter username and password for new user.
You have 9 options
  1. Add user to denied list
  2. Remove user from denied list
  3. Print user from denied list
  4. Search user
  5. Set permission for user
  6. Print the permission of user
  7. Check the permission of user
  8. Add new user to list
  9. Quit
